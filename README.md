# PasturePay

PasturePay is a decentralized livestock-backed micro-lending platform built on the Stellar blockchain using Soroban smart contracts. The platform enables farmers and livestock owners to tokenize real-world livestock assets as NFTs and use them as collateral to access instant USDC loans.

The protocol is designed to improve financial inclusion for underserved agricultural communities by creating a transparent, secure, and decentralized lending infrastructure. Farmers can register livestock, mint collateral-backed NFTs, request loans, and manage repayments directly on-chain without relying on traditional banking systems.

## Features

* Livestock NFT minting on Soroban
* Collateral-backed USDC loan issuance
* Automated interest and repayment logic
* Loan tracking and repayment management
* Liquidation engine for defaulted loans
* Farmer and lender dashboards
* Wallet authentication with Freighter SDK
* Real-time loan and collateral analytics
